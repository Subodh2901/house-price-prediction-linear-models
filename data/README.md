## Dataset

The dataset used in this project was obtained from Kaggle and is licensed under the Apache License 2.0. It contains real estate property information including structural attributes, amenities, location-related features, and sale prices.

The dataset is included in this repository to support reproducibility and ease of execution.

**Source:** Kaggle  
**Dataset:** Real Estate Dataset  
**Author:** oxcolaa  
**License:** Apache License 2.0  
**Link:** https://www.kaggle.com/datasets/oxcolaa/real-estate-dataset/data

**Local path:** data/real_estate_dataset.csv

The dataset was used in its original form without modification.
